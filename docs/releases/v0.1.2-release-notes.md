## What's New in v0.1.2

**🏠 Extended room shapes** — two new geometry types join the rectangular room calculator: **L-shaped rooms** (two-section composite) and **circular rooms** (full circle area and circumference). This release rounds out the core calculation engine.

### ✨ New in v0.1.2

| Category | Feature |
|----------|---------|
| **L-shaped Rooms** | Two-section rectangular rooms calculated as sum of both sections; shared height, independent length/width; 6-segment outer perimeter displayed |
| **Circular Rooms** | Full circle area (πr²) and circumference (2πr); diameter shown as read-only derived value |
| **Shape Selector** | 3-way toolbar toggle: Rectangular / L-shape / Circular |
| **Shape-specific UI** | Input panel dynamically shows shape fields: Section A/B for L-shape, Radius for circular |
| **Dynamic Result Summary** | Result card adapts per shape type ("Circumference" for circular, "Outer perimeter" for L-shape) |
| **History Preservation** | Calculation history now stores shape type and restores all shape-specific fields |

### 🔧 Improvements

- Toolbar 3-way shape toggle replaces single rectangular mode
- Result summary now labels outputs by shape type

## Full Changelog

Since v0.1.0:

### Added
- **L-shaped room support** — two-section rectangular rooms calculated as sum of both sections; shared height, independent length/width per section; 6-segment outer perimeter displayed
- **Circular room support** — full circle area (πr²) and circumference (2πr); diameter shown as read-only derived value
- **Shape selector toolbar** — 3-way toggle: Rectangular / L-shape / Circular
- **Dynamic input panel** — shape-specific fields shown based on selected shape type
- **Shape-aware result summary** — "Circumference" label for circular, "Outer perimeter" for L-shape, "Perimeter" for rectangular
- **Shape type in history** — calculation history now records shape type and restores shape-specific fields on click
