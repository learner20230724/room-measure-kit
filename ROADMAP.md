# Roadmap

> What's next for room-measure-kit.

Checked items are shipped. Unchecked are ideas or in-progress.

---

## M1 — Core Calculations ✅

- [x] Rectangular room floor area (length × width)
- [x] Room perimeter calculation
- [x] Wall area estimation (with configurable wall height)
- [x] Flooring requirement with waste percentage
- [x] Paint usage from coverage rate
- [x] Metric / Imperial unit toggle

## M2 — UX & Accessibility ✅

- [x] Built-in room preset chips (quick demo / screenshot)
- [x] One-click result summary copy
- [x] Input validation and sensible defaults
- [x] Responsive layout (mobile-friendly)

## M3 — DX & Code Quality ✅

- [x] Vitest test suite for core calculation logic
- [x] TypeScript throughout
- [x] ESLint linting
- [x] Vite build with type checking
- [x] GitHub Actions CI (validate workflow)

## M4 — Deployment & Distribution ✅

- [x] GitHub Pages static deployment
- [x] Social preview SVG (Open Graph / Twitter Card)
- [x] Bilingual README (EN + ZH)
- [x] MIT License

## M5 — Sharing & Persistence (ideas)

- [x] Shareable URL state (encode inputs in query string)
- [ ] Calculation history (localStorage)
- [ ] Export result summary as PNG or PDF

## M6 — Extended Room Support (ideas)

## M7 — Theming & Polish (ideas)
